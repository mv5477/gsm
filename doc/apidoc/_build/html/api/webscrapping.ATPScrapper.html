ATPScrapper¶

class webscrapping.ATPScrapper¶

Functions to scrap and parse data from the ATP website

The results on this website are more complete, better formatted, and easier to parse than on Wikipedia Unfortunately, this site blocks web scrapping tools, so we could not use it Keeping this code just in case it becomes possible through some other way
