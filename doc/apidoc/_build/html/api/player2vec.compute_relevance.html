compute_relevance¶

player2vec.compute_relevance(edges: list, feats: DataFrame, weights: list) → dict¶

Cleaner/more general version of ‘compute_players_relevance’

Can be used for both players and contexts Supports case where no weights are used Used in v2

Parameters:

  • edges – list of closest neighbors

  • feats – DataFrame with players features

  • weights – features weights

Returns:
